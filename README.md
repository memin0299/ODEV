Bu proje kapsamında Android platformu için geliştirilen Pomodoro Tekniği Zaman Yönetimi Uygulaması, kullanıcıların odaklanarak verimli çalışmasını amaçlamaktadır. Uygulama, Pomodoro tekniğine uygun olarak kullanıcının belirlediği dakikalık çalışma ve kullanıcın belirlediği dakikalık mola sürelerini zamanlayıcı yardımıyla yönetir. Hedef kullanıcı kitlesi, ders çalışan öğrenciler ve zaman yönetimini geliştirmek isteyen bireylerdir..
Amaç:Kullanıcının Pomodoro tekniğini kullanarak verimli çalışmasını sağlamak.
